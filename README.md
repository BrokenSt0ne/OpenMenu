# OpenMenu
OpenTag Mod menu, works in the christmas beta!

HOW TO INSTALL:
1. Install BepInEx.
2. Start OpenTag then quit out.
3. Place OpenMenu1.0.dll in BepInEx/plugins.
4. Start the game and hold X.
